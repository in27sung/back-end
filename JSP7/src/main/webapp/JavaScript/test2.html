JavaScript의 특징

1. 자바스크립트는 interpret 언어(1줄씩 위에서 아래로 순차적으로번역, 에러 발생시 다음코드 실행x) <-> 컴파일 언어

3. 객체 기반의 언어

4. 공개된 언어(Open Source Based)

5. 다양한 라이브러리를 제공

JS 사용문법

1. JS는 대/소문자를 구분
2. 모든 문장의 끝에는 semicolon(;)
3. double quotation == quotation
4. () Round bracket, {} Curly bracket, [] Square bracket
